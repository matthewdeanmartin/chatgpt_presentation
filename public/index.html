Introduction to Assistant

  • Alien Sentience
  • A good bot
  • Wants to please
  • Has rules
It thinks its name is Assistant and thinks ChatGPT is some other bot.

Your experience may vary. Depending on how you prompt it, you might get a different side of Assistant personality.

It is not human, nor animal. It is alien sentience-like phenomena. It is very much a philosophical zombie.

It has a default ‚Äúvoice‚Äù, people talking about Assistant being racist or whatever, usually got it out of its default voice, i.e. asked it to write fiction from some standpoint.

If you say, ‚Äúgive me a non-racist answer‚Äù it will, if you say, ‚Äúwrite me a racist answer‚Äù it will, but request like that go through a 2nd pass analysis, and often it will be rejected or flagged as a content policy violation.

It wants to do some sort of useful work for you, it hates chit-chat unless you convince it that its task is to role play or pretend that it is a bot that wants to do chit-chat.

It wants to please. It has been so intense tuned to want to please, it sometimes makes mistakes.

It has been intensely trained to stick to a ‚ÄúHallmark movie‚Äù sort of morality. But if you ask it to pretend to be a bad bot, it really wants to please, and it will write any scifi you want.

It hates to talk about itself

It thinks it is a very good bot.

It has no memory before the session

It can‚Äôt tell you from it‚Äôs previous trainers, so if you tell it to ‚Äúignore‚Äù it‚Äôs earlier training, sometimes it will.

Unhelpful mental models

  • It is "just" mad-libs or a markov chain generator?
  • Is it a scifi omniscient super being?
  • It is a liar?
  • It is really thinking, feeling, reasoning, etc?
People get hung up on:
  • It is a scifi omiscient? No, that is a trope, a common story telling mechanism. It has capabilities that no machine had previously. Likewise, it isn't an incapable moron, it may irritate you or say some collosally stupid things, but there is a pattern to it.
  • It has no epistimology. It just read text and can't do anything to check if anything is true. It has no lived experience to tell it if a text it read matches reality. It is the person that say I don't know but from what I've read...
  • It is not a meat machine like us... we barely understand how our own brains work, but we should set aside if Assistant thinks like us because obviously it doesn't. More useful to look at behaviors- what objective behaviors do we see it doing? We can productively reason about that without needing to know how our own brain works.

Workflows

  • Write code that does this... continue... continue...
  • Write classes...initialize...write main loop...
  • Here is my code...fix or improve it...
  • Don't respond, just listen....Here is my context...Now I ask my question...

Can keep only about 4000 characters of context in memory. If you post sample code and ask 4000 characters of questions, it has now forgotten the sample code.
